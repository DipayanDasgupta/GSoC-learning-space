# Coalition Stability Tracker

**Proposal connection:** Pillar 1 (Production Hardening) — extended demo

Tracks coalition birth, lifetime, and dissolution across 30 simulation steps.
Adds `merge()` as the fourth lifecycle method alongside join / leave / dissolve.

**Key outputs:** DataCollector tracks coalition count, free workers, average
coalition size, and average coalition skill at every step — ready to plot.

**Run:** `python models/coalition_stability/model.py`
